# BlackLab — AI 智能英語學習平台

一個以 AI 驅動的英語學習應用程式，透過即時生成個人化題目，讓使用者在遊戲化的段位系統中逐步提升英語能力。

## 功能模組

- **單字填字** — 看中文意思填入正確英文單字（附音標與例句）
- **文法練習** — 依段位出文法選擇題（附詳細中文解析）
- **聆聽辨識** — AI 真人發音，從相似選項選出正確單字
- **拼字挑戰** — 聽單字寫出完整拼寫（附提示功能）
- **AI 閱讀** — AI 即時撰寫全新文章 + 3 題理解問題
- **AI 寫作** — 寫一段英文，AI 給予評分與詳細批改報告

## 段位制度

**35 級進階之旅：
青銅（Lv.1-5）→ 白銀（Lv.6-10）→ 黃金（Lv.11-15）→ 白金（Lv.16-20）→ 鑽石（Lv.21-25）→ 大師（Lv.26-30）→ 菁英（Lv.31+）**

**六大屬性追蹤：詞彙力、文法力、聆聽力、拼字力、閱讀力、寫作力**
